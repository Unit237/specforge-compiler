# Hello World

A tiny Python CLI that prints a greeting.

## Behavior

- Running `hello` with no arguments prints `Hello, world!` and exits 0.
- Running `hello <name>` prints `Hello, <name>!` and exits 0.
- Running `hello --help` prints a short usage message.

## Constraints

- One file only: `hello.py`.
- Pure standard library — no external dependencies.
- Python 3.9+ compatible.
